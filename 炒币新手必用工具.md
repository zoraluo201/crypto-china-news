# 新手小白炒币必备的工具

今天来跟大家聊一聊新手炒币必备的工具箱。我们来看一下新手炒币需要具备哪些工具。

# **一、交易平台**

**1.完全小白**

建议你先使用中心化的交易所，比如币安、OKX、火币等。这三大所对新手的使用习惯还是比较友好的，大家可以根据自己的情况来选择。炒现货建议还是用币安，因为它的赚钱效益最好。

**2.中阶玩家**

中阶玩家和中高级玩家都使用自己的钱包。比较典型的是TP钱包，你可以自己百度一下，查找TP钱包的用法，用钱包来保管自己的资产。

**3.小白和中高级玩家的区别**

数字货币的资产在中心化交易所（如币安、欧易），相当于交易所这个平台帮你保管数字资产，你的币和资产都在交易所的钱包里。而你自己创建TP钱包，相当于你的数字资产放在自己的钱包里，由自己保管。钱包的操作比中心化交易所稍微复杂一点，但习惯之后，两者之间也没什么区别。

# **二、看行情软件**

**1.初阶小白**

可以使用AICOIN看盘软件，它有手机版本和电脑版本，你可以都安装一下。里面可以设置自动分组和自己的自选，一些关注的币种可以列入自选。

**2.中高阶玩家**

中高阶玩家使用专业的看盘软件，一般自带编程功能，你可以自己编写一些指标、代码、信号等来辅助看盘。

# **三、看资讯软件**

**1.新手小白**

你可以看资金财经、非小号，了解代币的基本情况，比如流通量、价格、化形量，以及它目前在哪些地方可以买到。

**2.中高阶玩家**

建议使用专业的数据服务商，查看行业数据，通过行业的发展状况以及资金链的情况，了解当下现状。一般看Twitter等国外社区行业大咖对行业和项目的点评观点，他们的观点基本代表了行业最新的热点和情况。同时也可以自己深入研究项目，进行长期跟踪。

以上三大块就是我们小白朋友从零基础到中高阶玩家的一个过程。这些工具也在不断变化，多掌握一个技能，就能对市场有更多应对的可能。
