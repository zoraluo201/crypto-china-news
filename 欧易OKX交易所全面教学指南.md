# 欧易OKX交易所完整教学指南

近期在台湾，一家交易所频繁出现在各种场合，如篮球场、公交车和线下活动，它就是OKX。许多人可能对OKX感到好奇，它究竟是何方神圣。本文旨在揭开这个老牌交易所的背景和特色。投资加密货币交易所仍存在风险，切记分散资产，避免全仓投入。现在，我们立即开始介绍。

**OKX交易所是什么？**

如果你是2021年前入场的币圈老玩家，会发现OKX的前身是OKEX。随着更多元素加入，为探索无限可能，2022年品牌正式更名为OKX。

**OKX交易所到底安不安全？**

OKX采用多签名技术保护钱包储存，并定期进行安全审计。此外，它公布储备金证明，确保资产透明度。后续成立了欧易万全基金，持续拨出平台收益，以在安全风险时保障用户资产。整体而言，撇除政治和监管风险（多数交易所共有的问题），其安全保障相当充分，值得用户信赖。但用户仍需留意最新风险动态。

**现在市面上的交易所有这么多，为什么我们要选择OKX呢？**

根据CoinMarketCAP数据，OKX的现货交易量位居第七，处于前十地位。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4da5b6c5b4_1724843474-47.jpg) 

在衍生品领域，它排名第三。因此，OKX是一个规模庞大的老牌交易所。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a7527a1a9a_1724843516-48.jpg) 

除规模外，其交易界面对新手非常友好。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8fcc762f53_1724843833-49.jpg) 

OKX还提供广泛的加密货币交易和理财产品供用户选择。最重要的是，它拥有自己的Web3钱包，便于用户在Web2和Web3世界间切换，这是一个独特功能。整体上，OKX是加密货币市场老用户值得信赖的平台。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/820c2b5d54_1724845226-50.jpg) 

早在2017年，市场就将币安、火币和OKEX合称为HBO，是当时的三巨头之一。OKX的平台币OKB也表现良好，表明投资者对其信任。了解OKX的安全性和特色后，接下来介绍注册流程。

# **入金USDT**

在“买币”区域选择“快捷买币”。银行卡方式最快，但收取刷卡手续费。需评估信用卡回馈是否足够。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/019bdcd9ea_1724846458-52.jpg) 

**2.P2P交易**

P2P交易是直接向加密货币卖家购买，OKX作为平台保障双方权益。优点是付款方式多元，缺点是店家和汇率较混杂。建议选择认证店家。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1fbf636ba0_1724846815-53.jpg)

# **币币交易**

入金完成后，可开始交易。点击首页上方【交易】，包含基础交易、衍生品、策略、跟单和模拟交易等。若无资金，可使用模拟交易模式。现货交易介绍：点击【基础交易】中的【币币】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/b78cb29f47_1724849736-61.jpg) 

界面看似复杂，但操作简单。从左开始：左侧选择交易种类和币种，如持有USDT购买BTC，选“BTC/USDT”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/743e591bb0_1724850048-62.jpg) 

中间是币价走势图，可切换时间如5分钟、1小时等。加密货币中，绿色代表上涨，红色代表下跌。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4129b09dfa_1724850212-63.jpg) 

右侧是市场挂单簿，显示当前挂单深度。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e3d991f583_1724850375-64.jpg) 

下单在中下方，可选限价委托、市价委托或止盈止损。“市价单”以当前价格立即成交；“限价单”在特定价格执行；“止盈止损”预设出场和止损点。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3575244972_1724850608-65.jpg) 

开单后，资料显示在最下方。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/03a9fe23c0_1724850722-36.jpg) 

例如，限价29000买入比特币，输入数量后点击【买入BTC】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7fe80f7558_1724850882-37.jpg) 

下方显示订单状态，如不想成交可【撤单】。现货交易整体简单。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/cfd918f18b_1724851046-38.jpg) 

# **合约交易**

若想玩衍生品（合约交易），点击上方【交易】，在衍生品选项中有永续、交割、杠杆和期权。“永续合约”是币圈特色产品；“交割”是传统季度合约；“杠杆”类似借贷交易；“期权”是传统选择权。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/dcdab6880f_1724851493-39.jpg) 

以最常见的永续合约为例。永续合约无到期日，与季度合约不同，只要不强制平仓，可永久持仓。但需确保合约价格不脱钩现货价格。

上方显示指数价格（当前现货价）和标记价格（永续合约价）。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/cf38079c9c_1724852106-40.jpg) 

为防止脱钩，平台建立资金费率机制。右上角可见费率：正数表示多头仓位多，多头需支付空头费用；负数表示空头仓位多，空头需支付多头费用。这是币圈特殊玩法。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3702b0ce64_1724852388-41.jpg) 

交易界面类似现货，主要区别在下单区。有全仓和逐仓选项：“全仓”以全部仓位为保证金，爆仓点较远，但爆仓损失全部保证金；“逐仓”可调整保证金，爆仓点较近，但损失部分保证金。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ece6c7c066_1724852870-42.jpg) 

右侧显示杠杆倍数，倍数越高风险越大。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/79946b1b30_1724852997-43.jpg) 

下方有计算器，可估算爆仓价格评估风险。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/bcfd0df77b_1724853120-44.jpg) 

操作示范：用市价委托买入做多，选择USDT，输入10U数量，全仓1倍买入。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/22a5a977e2_1724853346-45.jpg) 

下方显示当前仓位：持仓量、开仓均价、保证金、标记价格和预估强平价（爆仓价）。开单初期可能未显示，正常会显示爆仓价格。后方是收益和收益率。如想平仓，可选平仓或市价全平。操作类似现货，但需注意仓位和杠杆倍数。开单前务必计算爆仓价格。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/5005eb1be3_1724853755-46.jpg) 

# **策略交易**

若无时间主观交易，可在【交易】中找到策略交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4520fb779a_1724854402-47.jpg) 

可创建自定义策略或使用他人策略。适合无法频繁操作的用户。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0d30f4fb56_1724854722-49.jpg) 

若策略复杂，可选择跟单模式。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e8be80b471_1724854753-48.jpg) 

OKX提供跟单功能，让缺乏经验的用户复制专家交易。界面显示多个专业交易员，建议选择收益高者。胜率不一定准确，因可能涉及扛单或回撤。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e4f5000c79_1724855045-50.jpg) 

点击查看整体收益率再决定跟单。建议杠杆勿过高，设置止盈止损、每日本金上限和保证金费用。交易员风险承受不同，熊市下保护本金优先。跟单时务必设止损点，避免资金损失。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4d5c0d18af_1724855386-51.jpg) 

# **Web3钱包**

除交易外，OKX的特色是Web3钱包。点击最上方【Web3钱包】，进入后点右上角“链接钱包”。可选择OKX Web3钱包或其他区块链钱包连接。以OKX钱包为例，需安装插件。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/89bbcffcf9_1724855708-53.jpg) 

进行安装。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e052b905aa_1724855770-54.jpg) 

完成后，右上角弹出选项，选“我没有钱包”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/c4e35d2673_1724855847-55.jpg) 

然后注册。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8ce2b81055_1724855897-56.jpg) 

钱包设定后，如未备份需进行备份。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8ba5483c5b_1724855962-57.jpg) 

若新手觉得MetaMask复杂，可试用OKX钱包。不习惯也可连接其他钱包。

Web3钱包可查看所有资产，包括币种和NFT。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8821279f30_1724856169-59.jpg) 

可直接进行交易，如swap链上操作。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8fe81e4b9a_1724856289-60.jpg) 

挖矿或质押在“赚币”区域聚合，提供多种币种投资、矿池或机枪池，节省时间。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ccc372acfa_1724856380-61.jpg) 

还提供NFT市场聚合。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/23de846c3c_1724856483-62.jpg) 

整体为一站式服务，简化链上操作。嫌麻烦的用户可直接使用。

返回交易所界面，点击【金融】使用其他服务，如赚币、借贷或Jumpstart（IEO平台）。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/bc82ad342e_1724856757-63.jpg) 

若为开发者，在“公链”探索OKBC链上资讯。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1a60e73e5b_1724856867-64.jpg) 

另一特色是子账户功能。点击右上角头像，选“子账户”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/72d563b32b_1724856983-65.jpg) 

用于分隔不同交易账户，如跟单、策略、合约或现货。也适合空投活动，避免地址追踪。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/568340dfcd_1724857176-66.jpg) 

总结：OKX是多样化加密交易平台，提供多功能增强用户体验。改名后品牌曝光和形象更显质感沉稳。但提醒用户，加密货币交易有风险，切勿全仓投入，务必分散风险。
