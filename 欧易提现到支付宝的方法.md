# 欧易如何提现到支付宝？

欧易OKX是全球知名的数字货币交易所，主要为全球用户提供比特币、莱特币、以太币等数字资产的现货及衍生品交易服务。在进行交易时，如何将资金提现到支付宝呢？下面将为你进行详细演示。

# **第一步：欧易APP下载**

**1) 安卓手机下载**

安卓用户可访问 [https://vlink.cc/okxchina](https://vlink.cc/okxchina) 下载欧易安卓版APP。

**2) 苹果手机下载**

1. 苹果手机用户需先注册一个外区的Apple ID，可参考下方视频进行注册。若操作不成功或嫌麻烦，也可在百度上购买一个外区Apple ID账号，仅需几块钱。

2. 获取海外Apple ID后，打开App Store，点击右上角的小头像，向下滑动至底部，点击【登出】，退出原有ID。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/07c84de9ce_1709535360-47.jpg) 

3. 输入已注册的海外Apple ID，重新登录。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4579185041_1709005874-10.jpg) 

4. 在搜索栏输入“OKX”，第一个显示的是欧易的广告。选择下方的欧易进行下载，等待安装完成即可。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/246f86b818_1712376059-18.jpg) 

# **第二步：欧易账户注册**

1. 打开欧易APP，点击【注册/登录】，选择“立即注册”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/531e934bc1_1712376358-21.jpg) 

2. 输入邮箱地址，填写邀请码 **76527935**，使用此邀请码可享受20%手续费折扣。点击【注册】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/c3e3e16d12_1712376529-22.jpg) 

3. 进行真人验证，根据提示依次点击下方图标，再点击【确定】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d7e98032d2_1711068832-22.jpg) 

4. 输入发送至邮箱的6位验证码，点击【下一步】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/099db2dde4_1712376894-23.jpg) 

5. 居住国家选择【中国】，勾选创建账户的服务条款，点击【下一步】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/16da1cdec6_1711068996-24.jpg) 

6. 设置账户密码，点击【下一步】，欧易账户注册成功。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/29099ff437_1712377414-24.jpg) 

# **第三步：欧易身份认证**

1. 打开欧易APP，点击【去认证】，然后选择【立即认证】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/bac49d9fdf_1711069888-28.jpg) 

2. 居住国家或地区选择“中国”，证件类型选择“身份证”，点击【下一步】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/2127ec2d3f_1709544254-21.jpg) 

3. 输入真实姓名，证件类型选择“身份证”，输入身份证号，点击【下一步】。

需要注意的是，此处必须填写真实的身份信息，否则后续的人脸识别将无法通过。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e6d1f1c31f_1709545548-25.jpg) 

4. 仔细阅读下方红框内的证件上传要求，准备好身份证件，点击【开始】按钮。根据系统提示扫描身份证正反面，随后进行人脸识别（包括点头、眨眼等动作）。完成以上步骤后，认证即告完成。稍等几分钟，等待认证通过。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8b15d41167_1709544355-22.jpg) 

# **第四步：账户安全设置**

1. 返回欧易首页，点击左上角的九个点，再点击“个人资料和设置”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7af9ba9e2b_1712379607-29.jpg) 

2. 点击【安全设置】，选择【安全中心】。分别对身份验证应用、手机号和邮箱进行设置。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6e31874783_1712379768-30.jpg) 

# **第五步：购买USDT**

1. 点击上方的【买币】按钮，选择【C2C交易】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/69b89206e6_1711070381-34.jpg) 

2. 右上角的CNY代表人民币，即我们要使用的法币。这里可以用人民币购买USDT，并且可以筛选支付方式。点击右侧的【筛选】，支付方式包括银行卡、支付宝和微信，可全部勾选，点击【确认】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e9841b54e2_1711070745-35.jpg) 

3. 币种选择USDT。筛选所需购买金额，例如想买100元，则输入100。

下方的商家“木子梅旗舰店”成交量和成交率都很高，可选择与其交易，点击后面的【购买】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e232110e93_1712380738-31.jpg) 

4. 再次输入要购买的USDT金额，下方会计算出购买100元可获得13.55个USDT。点击【0手续费购买USDT】，再点击【获取付款详情】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/c8773549c4_1712381007-35.jpg) 

5. 根据页面中商家展示的付款方式完成付款，付款完成后点击【我已支付】，等待卖家放币。

* 完成付款后再点击“我已付款，通知卖家”，未付款时请勿点击；
* 不要在聊天框内与卖家发送任何关于买币的问题；
* 为避免账户被冻结，请务必使用实名付款；
* 付款前需核实收款人与订单信息是否一致。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/437370417b_1712381217-36.jpg) 

# **第六步：USDT提现到微信**

1. 回到欧易首页，点击【买币】，选择【C2C买币】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/fcc9fd1884_1712465625-69.jpg) 

2. 上方选择【出售】，币种选择“USDT”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/39c2df6d69_1712465718-70.jpg) 

3. 点击【金额】，输入想要出售的USDT数量，点击【确认】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/671732192d_1712465799-71.jpg) 

4. 点击【支付方式】，选择“支付宝”，点击【确认】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0493e50941_1712465946-74.jpg) 

5. 接下来筛选商家。注意，应选择注册时间在1年以上、成交量和成交率均较高的商家进行交易。

例如“恒飞诚信商贸”这家，成为商家已超过一年，成单数超过一万单，成交率高达99.98%，非常可靠。选择与其交易，点击【出售】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7ae4f3f533_1712466279-77.jpg) 

6. 输入要卖出的数量，例如输入20，下方会计算出将收到146.6人民币。点击【0手续费出售】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/82bcf332f8_1712466555-78.jpg) 

等待买家付款。当支付宝账号收到买家付款后，点击已收款，并在规定时间内将USDT放给对方。至此，完成整个提现人民币到支付宝的流程。
