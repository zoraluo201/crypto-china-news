# **二、购买USDT**

如何使用欧易交易所购买比特币？进入APP后，第一步不是直接购买比特币，而是先购买USDT。

**为什么要先买USDT呢？**

这好比进入赌场，你需要先将现金兑换成筹码，再用筹码参与各种游戏。USDT就是数字货币世界中的筹码。USDT是一种稳定币，与美元1:1挂钩，但存在千分之几的微小汇率浮动。它由泰达公司发行，因此又称泰达币。稳定币还有其他种类，如USDC，但USDT是我们最常用的稳定币。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/efbe5aedd7_1712553293-49.jpg) 

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4a0fa0f99a_1712553298-50.jpg) 

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/23cab4015f_1712553515-53.jpg) 

**为什么会有稳定币这种东西呢？**

这是因为目前全球许多国家不认可加密货币的合法地位，加密货币在很多地方处于灰色地带，不允许用法币直接交易加密货币。因此，稳定币应运而生，它介于法币与加密货币之间。通过法币购买稳定币，再用稳定币交易其他数字货币，即可实现交易。于是，便形成了当前这种局面。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/56df96089c_1712553792-54.jpg) 

**在欧易里应该怎么买USDT呢？**

简单来说，通过欧易的快捷买币或C2C买币功能，平台会为你匹配交易商家。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7c8b4230d1_1712554397-64.jpg) 

然后，根据商家要求的收款信息进行付款，可通过支付宝、微信或银行完成。付款后，平台会将商家的币释放给你。欧易在此过程中并不参与交易，仅起到类似支付宝的撮合作用，帮助双方完成买U交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ebcdce03a0_1712554465-65.jpg) 

购买USDT后，USDT会存放在资金账户中。如需购买其他数字货币，需将资金账户中的USDT转移到交易账户。点击【资金划转】。

这里需要确认要用多少U购买比特币。例如，我想购买100U的比特币，则输入100，点击【确认】即可划转成功。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/5c604cef80_1712554808-66.jpg) 

查看交易账户，现在已增加100U。交易所通常设有资金账户、交易账户和金融账户三种账户，分别对应不同使用功能。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1ef549fd04_1712555010-67.jpg) 

# **三、买卖比特币**

现在交易账户中已有足够的U来购买比特币。如何进行购买？可以直接点击【闪兑】，闪兑功能允许你输入所需U，直接兑换成比特币。但这种方式不推荐，因为存在损耗，价格不理想。不过，如果你急于获得比特币，可以通过此方式操作。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0ef6eff077_1712555249-68.jpg) 

我们最常用的买币方式是交易。点击下方中间的【交易】，初次看到交易页面可能会感到困惑。我来讲解一些关键指标。首先是左上角的BTC，即比特币。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/098ae7864b_1724832285-61.jpg) 

你可能不知道想买的币的缩写是什么，可以前往“非小号”网站查询对应缩写。该网站提供各种数字货币的介绍，包括中文名、英文名、价格、市值等。找到所需数字货币后，用其代号在搜索框内搜索即可。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/151a198ca5_1712556409-71.jpg) 

例如，我们购买比特币，输入BTC。这里弹出两个选项，分别对应USDT和USDC计价。本文演示以USDT计价，点击“BTC/USDT”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ce133a566e_1712556592-73.jpg) 

进入用USDT购买比特币的页面。点击【买入】，对应右侧报价：上方红色为卖出价格，下方绿色为买入价格。中间红色大字为实时成交价，即比特币报价。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/66a43c1d35_1712556891-74.jpg) 

如需查看更多内容，可点击右上角小图标，进入K线图查看行情。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4903b58797_1712557090-77.jpg) 

选择下单方式，主要用到限价委托和市价委托。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6ce8ee8e57_1712557515-78.jpg) 

“限价委托”是指我们报一个价格，只有市场价格接近时才能成交。限价委托的好处是以理想价格成交，但缺点是市场不以个人意志发展，可能无法成交。

“市价委托”是系统按市场价自动完成交易。优点是快速成交且确保交易完成，缺点是系统匹配的价格可能不是最优。对于资金量较小的用户，选择市价委托即可。

这里以“市价委托”为例进行一笔交易，例如购买100 USDT的比特币。下方显示“最小下单金额0.5USDT”，即购买比特币只需超过0.5U即可。尽管比特币价格很高，但无需一次购买1个或0.1个比特币，最少0.5U就能购买，这一点非常友好。输入100U后，点击【买入BTC】立即下单成功，交易完成。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9ebd03e612_1712557786-79.jpg) 

点击右下角的【资产】，上方选择交易账户，即可查看购买的比特币。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/63df0fa8a2_1712557917-80.jpg) 

若想卖出比特币，操作类似。进入交易页面，点击【卖出】，例如想卖出所有BTC，将百分比拉满，点击【卖出BTC】即可将比特币换成USDT。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/014b902795_1712558218-81.jpg) 

# **四、简单赚币**

持有比特币时可以做些什么？点击【资产】。一种方式是利用欧易的简单赚币，让比特币产生年化约5%的收益。点击【金融账户】，向下滑动，点击“前往金融”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0b4050dd5b_1712558414-82.jpg) 

这里有一个BTC页面，点击进入。点击【简单赚币】，点击【最大】，将全部BTC投入其中。此处有限额，仅对0.03个比特币有平台补贴，年化收益5%。我的金额未达到限额，直接申购即可完成。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1c6367304b_1712558905-60.jpg) 

简单赚币相当于欧易的余额宝。其中的币可随时产生收益，需要使用时也可随时赎回，非常方便。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4d393d1934_1712559053-61.jpg) 

如果你购买的比特币金额较大，或对中心化交易所（如欧易、币安）不够信任（此前FTX曾爆雷），可以选择将比特币直接提取到冷钱包。你可以在“资金账户”中通过提币操作，将币转移到更安全的地方。或者，你也可以将比特币提到电子钱包或U卡等可信平台直接消费，省去将其卖成USDT再出金的步骤。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/deb36a54c3_1724834835-63.jpg)
